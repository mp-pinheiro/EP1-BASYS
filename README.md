# EP1-BASYS
Primeiro exercício Prático de Programação para a disciplina ACH2096.

## ORGANIZAÇÃO
O projeto está organizado em:
- **src**: arquivos do código fonte.
- **bin**: arquivos compilados.
- **doc**: documentação do software.

## COMPILING
```
$ make
```

## RUNNING
* ### Windows
	```
	$ run.bat
	```

* ### Linux
	```
	$ ./bin/BASYS
	```
